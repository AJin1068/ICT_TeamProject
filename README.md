## 🖥️ 개발 목표

---

교육 과정에서 학습했던 Spring에 대한 개념을 정리하기 위해 이 프로젝트를 시작했습니다.

주제는 시장성이 있는 P2P 중고 거래 사이트를 모티브로 하였고, 경매 기능과 가격 필터링 같은 기능을 추가하여 기존 사이트와 차별성을 두었습니다.

사용자의 편의를 가장 중점에 둔 우리는 상추 마켓 입니다.
